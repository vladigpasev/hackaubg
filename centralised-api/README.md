# Centralised API

NestJS service scaffolded for the HackAUBG workspace.

## Environment

Copy `.env.example` to `.env` and adjust values as needed:

```bash
PORT=3001
```

If `.env` is not used, you can still override the port inline:

```bash
PORT=3001 npm run start:dev
```

## Install and Run

```bash
npm install
npm run start:dev
```

The service defaults to port `3001`.

## Useful Commands

```bash
npm run build
npm run lint
npm run test
npm run test:e2e
```
